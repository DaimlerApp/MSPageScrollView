# MSPageScrollView
iOS横向多页滑动，多导航页切换，横向TableView，原生乐视视频频道页框架，类似网易新闻，腾讯新闻手机客户端等交互方式，已添加autoLayout自动布局，适应设备旋转，可用于iPhone，iPad软件开发。
